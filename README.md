# Finance Management System

## Overview

A Java-based Finance Management System that helps users manage income and expenses, track financial transactions, generate reports, and visualize financial data.

## Features

* User Login System
* Add and Manage Transactions
* Expense Tracking
* Dashboard for Financial Overview
* Report Generation
* Database Integration using MySQL
* Excel Report Export using Apache POI
* Data Visualization using JFreeChart

## Technologies Used

* Java
* MySQL
* Maven
* Apache POI
* JFreeChart
* Swing GUI

## Project Structure

* LoginPage
* Dashboard
* AddTransaction
* ReportsPage
* DBConnection
* Expense_Management
